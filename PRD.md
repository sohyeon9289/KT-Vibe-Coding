# PRD: 핀포커스 라이트 웹서비스

## 1. 개요
- **서비스 이름**: 핀포커스 라이트
- **서비스 유형**: 정보 제공형 금융 웹서비스
- **개발 환경**: Nest.js (백엔드), TailwindCSS (프론트엔드 UI), TypeScript (전체 언어)
- **구성 형태**: 일반적인 블로그 구조 (메인 페이지 + 글 상세 페이지)
- **목표**: 투자 상품의 수익률과 위험도 정보를 안전하고 신뢰성 있게 제공하여 개인 투자자가 올바른 판단을 할 수 있도록 돕는다.

---

## 2. 주요 사용자
- **개인 투자자**: 초보~중급 수준, 보수적 성향의 사용자
- **목표 사용자 행동**:
  - 메인 페이지에서 최신 금융 정보와 분석 글 확인
  - 글 상세 페이지에서 특정 투자 상품의 수익률·위험도 분석 내용 열람
  - 관심 있는 글을 저장하거나 알림 구독

---

## 3. 핵심 기능
### 메인 페이지
- 최신 글 목록 표시 (투자 상품 정보, 위험도 분석 글)
- 카테고리/태그 기반 필터링
- 검색 기능 (상품명, 키워드)

### 글 상세 페이지
- 투자 상품별 수익률 및 위험도 분석 결과 표시
- 그래프 및 표를 통한 시각화
- 출처 및 갱신 날짜 명시
- 관련 글 추천

### 사용자 기능
- 관심 글 저장 (북마크)
- 알림 구독 (새로운 글/업데이트 시 이메일 또는 푸시 알림)
- 사용자 성향 설문 기반 맞춤 글 추천 (향후 확장)

---

## 4. 데이터 흐름
1. **상품 정보 수집**
   - 금융기관 API 및 공시자료에서 데이터 수집
   - JSON/CSV 형태로 정제 후 DB 저장
2. **위험도 분석**
   - 과거 수익률 및 변동성 데이터 기반 위험도 계산
   - 결과를 등급 및 설명 텍스트로 변환
3. **글 생성**
   - 분석 결과를 블로그 글 형태로 자동 생성 또는 관리자가 작성
   - TailwindCSS 기반 UI로 표시
4. **사용자 상호작용**
   - 글 열람, 북마크, 알림 구독
   - 사용자 입력 데이터는 DB에 저장

---

## 5. 기술 스택
- **백엔드**: Nest.js (API 서버, 데이터 처리, 인증)
- **프론트엔드**: TailwindCSS + TypeScript (UI, 반응형 디자인)
- **DB**: PostgreSQL (상품 데이터, 사용자 데이터 저장)
- **배포 환경**: Docker 기반 컨테이너, 클라우드 호스팅 (AWS/GCP)

---

## 6. 예상 제약사항
- 금융기관 API 접근 권한 및 데이터 라이선스 필요
- 위험도 분석 정확도는 입력 데이터 품질에 따라 달라질 수 있음
- 실시간 데이터 반영 시 서버 부하 및 보안 고려 필요

---

## 7. 기대효과
- 투자 판단의 정확성 향상
- 정보 탐색 시간 절약
- 과도한 위험 회피 가능
- 신뢰성 있는 금융 정보 제공으로 사용자 만족도 제고

---

## 8. 향후 확장
- 개인 자산 연동 기능
- 투자 시뮬레이션 제공
- AI 기반 맞춤형 분석 및 추천
